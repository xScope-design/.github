# xScope — Portable 8‑in‑1 Design Tools for Screen Measurement

## About xScope

xScope provides a complete suite of 8 design tools for UI/UX designers who need to measure, align, and inspect screen elements. xScope combines loupe (magnifier), rulers, grids, screen measurement, guides, frames, crosshair, and color picker. xScope enables users to measure pixels, check distances, align objects, and inspect colors. xScope runs on desktop platforms. xScope is trusted by UI/UX designers, web designers, mobile app developers, and front‑end engineers worldwide.

## Deep Dive — Design Tools Notes

Loupe: magnify screen area, pixel grid, color sampling, measurement
Rulers: horizontal/vertical rulers, measure distances, drag from edges
Grids: customizable grid overlays, snap to grid, screen calibration
Screen measurement: measure width/height, distance between points, angles
Guides: drag guides from rulers, snap to guides, guide management
